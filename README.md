# 🏥 Hospital Emergency Room Dashboard (Power BI)

## 📌 Project Overview
This project focuses on analyzing hospital emergency room (ER) operations using an interactive Power BI dashboard.  
It provides insights into patient flow, wait times, and service efficiency to support data-driven decision-making and improve overall hospital performance.

---

## 🎯 Objective
- Monitor and evaluate emergency room performance  
- Analyze patient trends and visit patterns  
- Assess wait times and service efficiency  
- Optimize hospital resource utilization  

---

## 📊 Dataset Overview
The dataset contains detailed information related to emergency room visits, including:
- Patient ID  
- Demographics (Age, Gender, Race)  
- Admission Date  
- Wait Time  
- Department Referral  
- Admission Status  
- Patient Satisfaction Score  

---

## 📌 Key Requirements
- Track total number of patients visiting the ER  
- Analyze average patient wait time  
- Measure patient satisfaction levels  
- Monitor department referral trends  
- Evaluate admission vs non-admission rates  
- Analyze patient demographics  
- Identify peak hours and patient flow patterns  

---

## 🛠 Tools & Technologies Used
- Power BI  
- Excel  
- DAX (Data Analysis Expressions)  

---

## 📊 Key Analysis Performed
- Patient volume analysis (daily and monthly trends)  
- Wait time performance evaluation  
- Patient satisfaction tracking  
- Department-wise referral analysis  
- Admission status comparison  
- Demographic distribution analysis  

---

## 📈 Key Insights
- Peak patient inflow observed during specific time intervals  
- Significant variation in wait times across different periods  
- Certain departments show consistently high referral rates  
- Patient satisfaction trends highlight areas for service improvement  

---

## 🚀 Conclusion
The dashboard enables stakeholders to gain actionable insights into emergency room operations.  
It supports improved decision-making, enhances patient care, and helps optimize hospital efficiency.
